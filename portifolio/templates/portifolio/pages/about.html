Sobre o Projeto

Este projeto foi desenvolvido utilizando o framework Django, conhecido por sua robustez e simplicidade para criar aplicações web. O objetivo do projeto era criar uma plataforma para gerenciamento de tarefas, com funcionalidades de criação, edição, e exclusão de tarefas, além de autenticação de usuários.

Etapas de Desenvolvimento

  • Planejamento: Definição dos requisitos e escopo do projeto.
  • Configuração do Ambiente: Criação do ambiente virtual e instalação do Django.
  • Modelagem do Banco de Dados: Criação dos modelos e migrações para o banco de dados.
  • Desenvolvimento do Back-end: Implementação das views, URLs e lógica de negócios.
  • Desenvolvimento do Front-end: Criação das templates utilizando HTML, CSS e Bootstrap.
  • Testes: Execução de testes unitários e de integração.
  • Deploy: Implantação do projeto em um servidor de produção.

Tecnologias Utilizadas

  • Django: Framework web de alto nível para desenvolvimento rápido e limpo.
  • Bootstrap 5.3: Framework front-end para design responsivo e mobile-first.
  • SQLite: Banco de dados leve utilizado durante o desenvolvimento.
  • HTML5 e CSS3: Linguagens de marcação e estilo para criação das interfaces.
  • JavaScript: Linguagem de programação para adicionar interatividade.
